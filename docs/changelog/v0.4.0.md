# v0.4.0

> **BREAKING(内部结构)**:顶层包按语义域合并,**顶层子包 25 → 15**;改善阅读与导航。
> 顶层公开 API(`from lighttrain import ...`)**契约不变**;仅 `lighttrain.<子包>.…` 的**深层导入路径**变化。

## 概览

`lighttrain/` 顶层子包过多(其中一大批是 1–3 文件的小抽象包),严重影响阅读。本版按**实际代码 import 耦合**
(而非文档叙述)把"功能或目的相同"的包合并到同一语义域顶层包下,采用**子包重挂**(`git mv` + import 路径前缀改写),
**不改任何运行时逻辑 / 控制流 / 签名**,纯搬迁。**硬切换**:不保留旧 `lighttrain.<pkg>` 子路径 shim。

合并依据均为代码实证耦合(无循环依赖):`artifacts → prepgraph → data`、`engine → distributed`、
`diagnostics → minimal`、`lineage → prepgraph`。

## 顶层 API 契约不变

[lighttrain/__init__.py](../../lighttrain/__init__.py) 的 re-export 仅**改 import 来源路径**,
**导出符号名与 `__all__` 一字未变**——以下全部照旧可用:

```python
from lighttrain import (
    CheckpointManager, LineageStore, Signal, StepContext, LoggerBus,
    content_hash, migrate, register, RootConfig, Trainer, ParallelContext, ...
)
```

仅当代码直接深层导入旧子路径(如 `from lighttrain.checkpoint.manager import CheckpointManager`)时需按上表改为新路径。

## 合并后顶层(15 包)

`data`(+prepgraph +artifacts)、`engine`(+update_rules +checkpoint)、
`callbacks`(+logging +invariants)、`observability`(diagnostics 升格 +lineage +minimal)、
`optim`(+architectures)、以及保持不动的 `registry` / `config` / `cli` / `builtin_plugins` /
`utils` / `lab` / `models` / `eval` / `distributed` / `trainers`。

> `registry`(149 处 `@register` 枢纽)、`distributed`(被 `engine._context` 依赖)保持顶层不动。
> `builtin_plugins` 内部 27 子目录不重组,仅同步其引用被移动核心包的 import 路径。

## tests 与 docs 同步

- **tests**:全部测试 import 改为新路径;镜像测试目录随源重挂
  (`tests/prepgraph` → `tests/data/prepgraph`、`tests/checkpoint` → `tests/engine/checkpoint`、
  `tests/{diagnostics,lineage,minimal}` → `tests/observability/*` 等)。
- **docs**:更新所有非 deprecated 文档的模块/路径引用,含
  [architecture.md](../concepts/architecture.md) 的"五缝"路径,使文档与新结构对齐。

### 一处类型注解微调(非逻辑改动)

[observability/diagnostics/oom_report.py](../../lighttrain/observability/diagnostics/oom_report.py) 中
`torch.cuda.OutOfMemoryError` 的 `# type: ignore[attr-defined]` 因模块改名扰动 mypy 分析顺序而被判"冗余",
但其在 torch 旧版本仍**确实需要**(该处刻意不依赖 torch≥2.0 的 `OutOfMemoryError`)。故改为
`# type: ignore[attr-defined,unused-ignore]`,两版本环境皆静默,**运行时行为不变**。

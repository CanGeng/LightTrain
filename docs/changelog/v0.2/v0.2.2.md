# v0.2.2

## 概览

核心瘦身（轻：按 [DESIGN](../deprecated/DESIGN.md) §3.3「反向检查」纠偏）：把仍错置在 `lighttrain/` 的
**具体 frontier 实现**移入 [lighttrain/plugins/](../../lighttrain/plugins)，使核心/研究层与文档的三层架构对齐。

**relocate-only、零功能变化**：`import_all_components` 照常 walk `lighttrain.plugins`，这三组（纯 PyTorch、
无第三方依赖）继续默认自动注册；同一注册表、同一 recipe/CLI 行为。运行时一律经注册表按名解析，不直接 import
这些实现——故移动只触及各自包 `__init__` 的 re-export + 少量测试的直接 import。

## 移动（§3.3：specific objective / judge / 架构 adapter 实现 → frontier，协议留 core）

- **生成式 objectives** → [lighttrain/plugins/objectives/](../../lighttrain/plugins/objectives)：`diffusion` / `flow_matching` / `jepa`。
  基础 `next_token` / `masked_denoising` 留 `lighttrain.objectives`（v0.2.3 再下放）。
- **judges** → [lighttrain/plugins/judges/](../../lighttrain/plugins/judges)：`verifier` / `pairwise_llm`。
  Judge 协议留 [protocols.py](../../lighttrain/protocols.py)；运行时 `_build_judge` 经 `judge` 注册表类别解析。
- **jepa 架构** → [lighttrain/plugins/architectures/jepa.py](../../lighttrain/plugins/architectures/jepa.py)（与 rwkv/mamba/diffusion_unet 同处）。
  `ArchitectureProfile`/`ObjectiveProfile` 基建 + transformer profile 留 `lighttrain.optim.architectures`。

## 配套

- 移动文件内部相对 import 改绝对（`..protocols`/`..registry`/`.profile` → `lighttrain.*`），与现有 plugins 约定一致。
- 源包 `__init__` 去掉已移走模块的 re-export（[objectives](../../lighttrain/objectives/__init__.py)/[eval](../../lighttrain/eval/__init__.py)/[architectures](../../lighttrain/optim/architectures/__init__.py)）；~10 个测试 import 重定向到 `lighttrain.plugins.*`。

- 文档：[registry.md](../reference/registry.md) / [registry.zh-CN.md](../reference/registry.zh-CN.md) 把 jepa(model/objective) / 生成式 objectives / judges 标注为 plugin 并改文件路径。

## 账面修正（§3.3：builtin callbacks 属 research）

- `callbacks/builtins`（ema/best_ckpt/throughput/…）此前在 [check_core_size.py](../../scripts/check_core_size.py)
  `_CORE_DIRS` 中被计为 **core**，但 §3.3 判内置 callback 实现属 research（仅 `callbacks/base.py` 的 Callback 协议 +
  EventBus 是 core）。从 `_CORE_DIRS` 移除——纯账面（无代码搬迁；builtins 本就在 `lighttrain/callbacks/builtins/`、
  默认随包、auto-discovery 照常注册）。core 预算 **4871 → 4359 SLOC**（512 行计入 research）。
